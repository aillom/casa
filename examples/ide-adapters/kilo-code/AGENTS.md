# AGENTS.md

This project follows C.A.S.A - Context, Architecture, Stack & Automation.

Kilo Code should treat this as the primary project instruction file.

## Before coding

1. Read `casa.manifest.yaml`.
2. Read `CONTEXT.md`.
3. Read the relevant spec in `.casa/specs`.
4. Read the relevant policy in `.casa/kernel/policies`.
5. Use the smallest useful capability from `.casa/capabilities`.
6. Do not edit generated adapter files directly.
7. Run `npm run check` before completion when commands are available.
