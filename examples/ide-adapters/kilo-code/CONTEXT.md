# C.A.S.A Context For Kilo Code

## Source of truth

- `.casa/kernel`: principles, standards, policies and risk model.
- `.casa/specs`: feature, API, security and brownfield spec templates.
- `.casa/capabilities`: skills, subagents and workflows.
- `.casa/governance`: permissions, sensors, evals and audit guidance.
- `.casa/generated`: generated outputs. Do not edit directly.

## Rule

Write once in `.casa`, adapt everywhere through generated or tool-specific instruction files.
