# Workflow: Modernize Legacy Module With C.A.S.A

## Goal

Use this as a workflow card for brownfield modernization.

## Steps

1. Run read-only discovery.
2. Map entry points, dependencies, data and integrations.
3. Add characterization or smoke tests.
4. Create a seam or facade.
5. Introduce new behavior behind the seam.
6. Compare behavior and metrics.
7. Retire old code only after evidence and rollback planning.
