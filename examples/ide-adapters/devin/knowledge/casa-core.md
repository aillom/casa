# Devin Knowledge: C.A.S.A Core

## When to use

Use for any non-trivial task in a repository that follows C.A.S.A.

## Instructions

1. Read `AGENTS.md`.
2. Read `casa.manifest.yaml`.
3. Read the relevant spec in `.casa/specs`.
4. Read the relevant policy in `.casa/kernel/policies`.
5. Use the smallest useful capability from `.casa/capabilities`.
6. Do not edit generated adapter files directly.
7. Run `npm run check` before completion.

## Completion evidence

Report changed files, sensors run and remaining risks.
