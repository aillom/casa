# C.A.S.A Rule

This workspace follows C.A.S.A - Context, Architecture, Stack & Automation.

When answering or editing:

1. Read `AGENTS.md` and `casa.manifest.yaml`.
2. Use `.casa` as source of truth.
3. Follow relevant specs and policies.
4. Avoid editing generated adapter files.
5. Prefer small, testable changes.
6. Run or recommend `npm run check` before completion.
