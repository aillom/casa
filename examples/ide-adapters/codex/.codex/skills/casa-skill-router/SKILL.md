---

# Purpose

Choose the smallest set of capabilities needed for the task.

# Routing Examples

- UI work: `frontend-design-engineer`, `test-engineer`
- API work: `backend-clean-architect`, `api-contract-engineer`, `test-engineer`
- Security-sensitive work: `security-reviewer`, relevant implementation skill, `test-engineer`
- Legacy work: `legacy-archaeologist`, `test-engineer`

# Output

Return selected skills and why.
