# AGENTS.md

This project follows C.A.S.A - Context, Architecture, Stack & Automation.

## Trae usage

Use Trae workspace context and code indexing with this file plus `.agents/casa-core.md`.

Before coding:

1. Read `casa.manifest.yaml`.
2. Read the relevant spec in `.casa/specs`.
3. Read the relevant policy in `.casa/kernel/policies`.
4. Use the smallest useful capability from `.casa/capabilities`.
5. Do not edit generated adapter files directly.
6. Run `npm run check` before completion when shell access is available.
