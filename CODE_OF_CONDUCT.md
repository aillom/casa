# Code of Conduct

C.A.S.A is an open-source project by Aillom.

We expect contributors to communicate respectfully, assume good intent and focus on improving the project.

## Expected behavior

- Be respectful.
- Be constructive.
- Welcome newcomers.
- Discuss ideas, not people.
- Prefer evidence and reasoning over authority.

## Unacceptable behavior

- Harassment
- Personal attacks
- Discriminatory language
- Threats
- Spam
- Repeated bad-faith behavior

## Enforcement

Maintainers may remove, edit or reject comments, commits, issues and pull requests that violate this code of conduct.
