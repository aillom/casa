# Introduction

C.A.S.A — Context, Architecture, Stack & Automation — is an agent-native engineering control plane for software delivery.

It is designed for teams using AI coding agents such as Cursor, Codex, Claude, Devin, Antigravity and future tools.

C.A.S.A is not a template, coding agent or workflow builder. It is an architecture layer that provides reusable context, governance, agent capabilities, mission-based execution, modernization paths and automation.

It can be used as the architecture layer for governed vibe coding: fast AI-assisted implementation with stable context, explicit boundaries, reusable capabilities and validation sensors.

See `docs/agent-ide-examples.md` and `examples/ide-adapters` for examples across Codex, Cursor, Claude, Devin, Copilot and generic agents.
