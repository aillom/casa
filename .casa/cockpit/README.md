# C.A.S.A Cockpit

The cockpit is the future visual control plane for C.A.S.A.

It is not a workflow builder. It is an engineering cockpit for missions, context, agents, risks, evidence, modernization and adapter outputs.

## Initial Screens

- Mission Control
- Spec Canvas
- Agent Swarm Board
- Context Capsules
- Architecture Graph
- Legacy Modernization Map
- Risk & Policy Center
- Evidence Ledger
- Adapter Studio
- Docs

## Product Principle

C.A.S.A should make agent work visible, reviewable and evidence-driven.
