# Context Capsule: frontend-dashboard

## Use When

Building or changing dashboards, admin screens, SaaS UI, data tables, filters or reporting views.

## Load

- `.casa/capabilities/skills/frontend-design-engineer/SKILL.md`
- `.casa/capabilities/skills/test-engineer/SKILL.md`
- `.casa/kernel/standards/frontend.md`
- `.casa/governance/evals/ui-quality.eval.md`

## Watch

- Loading, empty, error and success states.
- Responsive layout and accessibility.
- Domain-specific copy and information hierarchy.
- API contract impact for displayed data.

## Validate

- UI tests or smoke checks when available.
- `npm run check`
- Screenshot evidence for visual changes when practical.
