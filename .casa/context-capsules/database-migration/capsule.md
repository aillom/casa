# Context Capsule: database-migration

## Use When

Changing schemas, migrations, indexes, tenant isolation, audit fields or sensitive data storage.

## Load

- `.casa/capabilities/skills/database-architect/SKILL.md`
- `.casa/capabilities/skills/test-engineer/SKILL.md`
- `.casa/kernel/policies/database-security.md`
- `.casa/kernel/standards/database.md`
- `.casa/governance/sensors/migration-risk.sensor.md`

## Watch

- Destructive migration risk.
- Backup and rollback plan.
- Tenant boundaries.
- Sensitive data ownership.
- Index impact for critical queries.

## Validate

- Migration risk gate.
- Integration tests when available.
- Rollback notes for high-risk changes.
