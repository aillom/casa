# Domain Map

## C.A.S.A Core

Defines stable method primitives: context, architecture, stack, automation, policies and risk model.

## Capability Layer

Defines reusable agent behavior through skills, subagents and workflows.

## Adapter Layer

Translates C.A.S.A Core into tool-specific files for Codex, Cursor and portable agent packs.

## Governance Layer

Defines permissions, protected paths, risk levels, sensors and review expectations.

## Modernization Layer

Defines brownfield discovery, safety baselines, seams, strangler migration and retirement guidance.
