# Playbook: Retirement

## Goal

Remove legacy code only after behavior, traffic and rollback requirements are satisfied.

## Steps

1. Confirm no active callers remain.
2. Confirm monitoring shows no usage.
3. Confirm rollback path or backup exists.
4. Remove code, docs and config together.
5. Update context maps.

## Output

- Retirement evidence.
- Cleanup PR.
- Updated maps and docs.
