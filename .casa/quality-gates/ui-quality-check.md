# Quality Gate: UI Quality Check

## Purpose

Validate user-facing interface quality.

## Applies When

- Screens, dashboards, forms, navigation or visual states change.

## Check

- Loading, empty, error and success states exist.
- Layout is responsive.
- Controls are accessible.
- Copy matches the product domain.
- Visual hierarchy supports repeated use.
