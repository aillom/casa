# Workflow: Implement Feature

1. Read spec.
2. Select skills with casa-skill-router.
3. Identify affected files.
4. Implement smallest safe change.
5. Add or update tests.
6. Update API contracts if needed.
7. Update docs if behavior changed.
8. Run validation.
9. Summarize risks and follow-ups.
