# Approval Matrix

## Low

- Agent may proceed with normal validation.
- Tests or docs updates are required when behavior changes.

## Medium

- Reviewer recommended.
- Run relevant sensors.
- Update specs, contracts and docs when applicable.

## High

- Human approval required.
- Security or specialist review required.
- Rollback plan required.
- CI gates must pass before merge.

## Critical

- Manual change window required.
- Two-person review required.
- Incident-style checklist required.
- Production rollback plan required before execution.
